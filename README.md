# YO-Planner

###### 계획적인 삶을 위해 필요한 planner

- 다양한 기능을 탑재한 플래너 앱

-------

### For

- 계획적인 생활을 위해 플래너가 필요

- 다양한 기능을 탑재한 플래너가 필요

사용자의 일정 관리 + 목적지 주변 검색
